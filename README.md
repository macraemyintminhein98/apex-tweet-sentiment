# APEX Tweet Sentimeter

**AI-Powered Twitter Sentiment Analysis**

This project provides a simple web tool to analyze the sentiment of recent tweets from a given Twitter URL or username. It leverages AI to classify tweets as positive, negative, or neutral, presenting the results with a clear visualization.

### Features
- Input Twitter URL or username.
- Fetches recent tweets.
- Provides sentiment breakdown (positive, negative, neutral).
- Simple pie chart visualization.

### Setup (Local Development)
1. Clone the repository.
2. Install dependencies (if using Next.js/React: `npm install`).
3. Configure your Twitter API keys and Sentiment Analysis API keys (e.g., Hugging Face).
4. Run the development server (`npm run dev`).

### Deployment
Designed for easy deployment on platforms like Vercel.

This `index.html` is a static placeholder. The full product would involve a Next.js frontend and Vercel serverless functions for API integration and sentiment analysis.
